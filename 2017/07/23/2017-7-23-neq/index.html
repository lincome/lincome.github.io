女儿情

July 23, 2017 98

鸳鸯双栖蝶双飞

满园春色惹人醉

悄悄问圣僧

女儿美不美

女儿美不美

说什么王权富贵

怕什么戒律清规

只愿天长地久

与我意中人儿紧相随

爱恋依 爱恋依

愿今生长相随

愿今生长相随

长相随

好听！
